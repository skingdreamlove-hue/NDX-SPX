# 美股情绪监测服务自启动配置说明

## 📋 问题描述
- **问题**: 电脑睡眠唤醒后，Flask服务进程被杀死，需要手动重启
- **目标**: 实现开机自动启动 + 睡眠唤醒后自动恢复
- **服务地址**: http://127.0.0.1:5000

## 🛠️ 解决方案概览

### 方案A：Windows任务计划程序（推荐）
- ✅ 系统级自动启动
- ✅ 支持睡眠唤醒触发
- ✅ 无需用户登录即可运行
- ✅ 最稳定可靠

### 方案B：守护脚本方案
- ✅ 实时监控服务状态
- ✅ 自动重启失败的服务
- ✅ 可安装到启动项

## 📁 已创建的文件

### 启动脚本
- `start_server.bat` - 标准启动脚本（显示命令行窗口）
- `start_server_silent.vbs` - 静默启动脚本（无窗口）

### 自动启动配置
- `create_windows_task.ps1` - Windows任务计划创建脚本
- `watchdog.ps1` - 守护监控脚本

## 🔧 实施步骤

### 第一步：配置Windows任务计划（方案A）

1. **以管理员身份运行PowerShell**
   ```powershell
   # 导航到项目目录
   cd "d:\web\美股情绪监测"
   
   # 执行配置脚本
   .\create_windows_task.ps1
   ```

2. **验证任务创建**
   ```powershell
   # 查看任务状态
   schtasks /query /tn "美股情绪监测服务"
   
   # 手动运行任务测试
   schtasks /run /tn "美股情绪监测服务"
   ```

### 第二步：配置守护脚本（方案B，可选）

1. **安装到启动项**
   ```powershell
   cd "d:\web\美股情绪监测"
   .\watchdog.ps1 -InstallToStartup
   ```

2. **手动运行守护脚本**
   ```powershell
   # 前台运行（显示日志）
   .\watchdog.ps1
   
   # 后台运行（静默模式）
   Start-Process powershell -ArgumentList "-WindowStyle Hidden -File .\watchdog.ps1"
   ```

## 🔄 故障排除

### 如果服务无法启动
1. **检查Python环境**
   ```powershell
   python --version
   python -c "import flask; print('Flask installed')"
   ```

2. **手动测试启动**
   ```powershell
   cd "d:\web\美股情绪监测"
   python web_server.py
   ```

3. **检查防火墙设置**
   - 确保Python有网络访问权限
   - 检查Windows Defender防火墙设置

### 如果任务计划不工作
1. **重新创建任务**
   ```powershell
   schtasks /delete /tn "美股情绪监测服务" /f
   cd "d:\web\美股情绪监测"
   .\create_windows_task.ps1
   ```

2. **检查任务日志**
   - 打开"任务计划程序"
   - 找到"美股情绪监测服务"任务
   - 查看"历史记录"标签页
