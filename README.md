# 美股情绪监测 - 桌面应用安装指南

## 功能特点
- ✅ 双击应用图标直接打开界面，无需服务器端口
- ✅ 点击"更新数据"按钮自动抓取最新数据
- ✅ 数据更新后自动刷新页面显示
- ✅ 完全本地运行，无需网络连接（除数据抓取外）

## 安装步骤

### 1. 安装 Node.js
- 访问 https://nodejs.org/ 下载并安装 Node.js (推荐 LTS 版本)
- 安装完成后，打开命令提示符，输入 `node --version` 确认安装成功

### 2. 安装依赖
在项目目录中运行：
```bash
npm install
```

### 3. 启动应用
双击运行 `launch_app.bat` 或执行：
```bash
npm start
```

## 打包成可执行文件

### 开发模式运行
```bash
npm start
```

### 打包成安装包
```bash
npm run dist
```

打包后的文件将在 `dist` 目录中生成：
- `.exe` 安装程序
- 可执行文件

### 创建桌面快捷方式
1. 打包后，将生成的 `.exe` 文件创建桌面快捷方式
2. 双击快捷方式即可直接运行应用

## 使用说明

1. **启动应用**：双击桌面快捷方式或运行 `launch_app.bat`
2. **查看数据**：应用启动后自动显示最新市场数据
3. **更新数据**：点击"更新数据"按钮，系统自动：
   - 执行 Python 脚本抓取最新数据
   - 生成图表和指标
   - 自动刷新页面显示最新结果

## 文件结构
```

├── launch_app.bat       # 启动脚本
├── index.html           # 主界面

```

## 注意事项

- 确保系统中已安装 Python 3.x
- 首次运行需要安装依赖，请保持网络连接
- 数据更新过程可能需要几分钟，请耐心等待
- 应用完全本地运行，数据不会上传到任何服务器
